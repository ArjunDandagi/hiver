# hiver


STEPS:

1. clone this repo with `gh repo clone ArjunDandagi/hiver` 

Don't have gh? install from [here](https://github.com/cli/cli#installation)

OR  

` git clone https://github.com/ArjunDandagi/hiver.git`

2. ` cd hiver` 

3. if you dont have boto3 installed please install via command `pip3 install boto3` 

3.1 ensure you have the default AWS credentials added in ~/.aws/credentials 

4. run the script with `python3 list_instances.py` 


or `chmod u+x list_instances.py && ./list_instances.py`
